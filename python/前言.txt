程序：算法 + 数据结构

Algorithm 算法
算法: 解决一个问题的完整方案，解决一系列问题的清晰指令

流程：
输入 ---> 算法 --->输出

Data Structures 数据结构
数据结构：计算机储存、组织数据的的方式，用于高效处理数据

好的算法：    快（运算速度快）    省（内存占用少）
对应两个指标： 时间复杂度         空间复杂度

在数据库中选用合理的数据结构来对数据进行操作
在服务器的带宽(数据传输通道最大容量)



学习算法：
1.理解算法思想，即运算逻辑、思维逻辑
2.在纸上模拟出实现流程
3.使用熟悉的编程语言来实现这个算法


数据结构和类型：
自定义的类可以理解为抽象的数据类型（ADT:Abstract Data Type）
同一种或同一类数据，具有相同的属性和方法
