# 顺序表

## 1. 线性表与顺序表的关系

线性表是由 `n` 个数据元素组成的有限序列，记作：

```text
L = (a1, a2, ..., ai, ..., an)
```

其中：

- `n = 0` 时，称为空表。
- `a1` 是第一个元素，也叫表头元素。
- `an` 是最后一个元素，也叫表尾元素。
- 除第一个元素外，每个元素有且只有一个直接前驱。
- 除最后一个元素外，每个元素有且只有一个直接后继。

顺序表是线性表的顺序存储结构，它用一段地址连续的存储单元依次存放线性表中的数据元素。

## 2. 顺序表的基本特点

顺序表的核心特点是：逻辑上相邻的元素，物理存储位置也相邻。

假设每个元素占用 `sizeof(ElemType)` 个字节，顺序表第一个元素的地址为 `LOC(a1)`，则第 `i` 个元素的地址为：

```text
LOC(ai) = LOC(a1) + (i - 1) * sizeof(ElemType)
```

如果采用 C 语言数组下标，则：

```text
第 1 个元素 -> data[0]
第 i 个元素 -> data[i - 1]
第 n 个元素 -> data[n - 1]
```

注意区分：

- 位序：从 `1` 开始，常见于教材描述。
- 下标：从 `0` 开始，常见于 C 语言数组。

## 3. 顺序表的优缺点

优点：

- 支持随机访问，可以通过下标直接访问任意元素。
- 存储密度高，不需要额外指针域。
- 结构简单，适合查找和遍历。

缺点：

- 插入和删除时需要移动大量元素。
- 静态顺序表容量固定，容易造成空间不足或空间浪费。
- 动态顺序表虽然可以扩容，但扩容需要重新申请空间并搬移元素。

## 4. 顺序表的存储结构

### 4.1 静态顺序表

静态顺序表使用固定大小的数组作为存储空间。

```c
#include <stdio.h>

#define MAXSIZE 100

typedef int ElemType;

typedef struct
{
    ElemType data[MAXSIZE];
    int length;
} SeqList;
```

含义：

- `data`：存放数据元素的数组。
- `length`：当前顺序表中的元素个数。
- `MAXSIZE`：顺序表最多能存放的元素个数。

### 4.2 动态顺序表

动态顺序表使用指针保存动态申请的连续空间。

```c
#include <stdio.h>
#include <stdlib.h>

#define INIT_SIZE 100

typedef int ElemType;

typedef struct
{
    ElemType *data;
    int length;
    int capacity;
} SeqList;
```

含义：

- `data`：指向动态申请的连续存储空间。
- `length`：当前元素个数。
- `capacity`：当前最大容量。

初始化示例：

```c
int InitList(SeqList *L)
{
    L->data = (ElemType *)malloc(sizeof(ElemType) * INIT_SIZE);
    if (L->data == NULL)
    {
        return 0;
    }

    L->length = 0;
    L->capacity = INIT_SIZE;
    return 1;
}
```

## 5. 基本操作

### 5.1 初始化

静态顺序表初始化时，只需要把长度置为 `0`。

```c
void InitList(SeqList *L)
{
    L->length = 0;
}
```

### 5.2 求表长

```c
int Length(SeqList L)
{
    return L.length;
}
```

时间复杂度：`O(1)`。

### 5.3 判断是否为空

```c
int IsEmpty(SeqList L)
{
    return L.length == 0;
}
```

时间复杂度：`O(1)`。

### 5.4 按位查找

按位查找是根据位序 `i` 查找第 `i` 个元素。

```c
int GetElem(SeqList L, int i, ElemType *e)
{
    if (i < 1 || i > L.length)
    {
        return 0;
    }

    *e = L.data[i - 1];
    return 1;
}
```

注意：位序 `i` 对应数组下标 `i - 1`。

时间复杂度：`O(1)`。

### 5.5 按值查找

按值查找是查找第一个值等于 `e` 的元素位置。

```c
int LocateElem(SeqList L, ElemType e)
{
    for (int i = 0; i < L.length; i++)
    {
        if (L.data[i] == e)
        {
            return i + 1;
        }
    }

    return 0;
}
```

返回值说明：

- 找到：返回元素位序。
- 没找到：返回 `0`。

时间复杂度：`O(n)`。

### 5.6 插入操作

在顺序表第 `i` 个位置插入元素 `e`。

插入前：

```text
L = (a1, a2, ..., ai, ..., an)
```

插入后：

```text
L = (a1, a2, ..., e, ai, ..., an)
```

代码：

```c
int ListInsert(SeqList *L, int i, ElemType e)
{
    if (i < 1 || i > L->length + 1)
    {
        return 0;
    }

    if (L->length >= MAXSIZE)
    {
        return 0;
    }

    for (int j = L->length; j >= i; j--)
    {
        L->data[j] = L->data[j - 1];
    }

    L->data[i - 1] = e;
    L->length++;
    return 1;
}
```

关键点：

- 合法插入位置是 `[1, length + 1]`。
- 插入第 `i` 个位置时，从最后一个元素开始向后移动。
- 移动方向必须从后往前，避免覆盖原数据。

时间复杂度：

- 最好情况：在表尾插入，`O(1)`。
- 最坏情况：在表头插入，`O(n)`。
- 平均情况：`O(n)`。

### 5.7 删除操作

删除顺序表第 `i` 个位置的元素，并用 `e` 返回被删除的值。

删除前：

```text
L = (a1, a2, ..., ai, ai+1, ..., an)
```

删除后：

```text
L = (a1, a2, ..., ai-1, ai+1, ..., an)
```

代码：

```c
int ListDelete(SeqList *L, int i, ElemType *e)
{
    if (i < 1 || i > L->length)
    {
        return 0;
    }

    *e = L->data[i - 1];

    for (int j = i; j < L->length; j++)
    {
        L->data[j - 1] = L->data[j];
    }

    L->length--;
    return 1;
}
```

关键点：

- 合法删除位置是 `[1, length]`。
- 删除第 `i` 个位置时，从第 `i + 1` 个元素开始向前移动。
- 删除完成后，表长减 `1`。

时间复杂度：

- 最好情况：删除表尾元素，`O(1)`。
- 最坏情况：删除表头元素，`O(n)`。
- 平均情况：`O(n)`。

### 5.8 遍历输出

```c
void PrintList(SeqList L)
{
    for (int i = 0; i < L.length; i++)
    {
        printf("%d ", L.data[i]);
    }
    printf("\n");
}
```

时间复杂度：`O(n)`。

## 6. 常见算法题

### 6.1 合并两个递增有序顺序表

题目：

已知两个顺序表 `A` 和 `B` 中的元素都按从小到大排列，设计算法将二者合并成一个新的递增有序顺序表 `C`。

思路：

- 使用三个下标 `i`、`j`、`k`。
- `i` 指向 `A` 当前元素。
- `j` 指向 `B` 当前元素。
- `k` 指向 `C` 当前插入位置。
- 每次比较 `A.data[i]` 和 `B.data[j]`，较小者放入 `C`。
- 某一个表扫描结束后，把另一个表剩余元素全部复制到 `C`。

代码：

```c
void MergeList(SeqList A, SeqList B, SeqList *C)
{
    int i = 0;
    int j = 0;
    int k = 0;

    while (i < A.length && j < B.length)
    {
        if (A.data[i] <= B.data[j])
        {
            C->data[k++] = A.data[i++];
        }
        else
        {
            C->data[k++] = B.data[j++];
        }
    }

    while (i < A.length)
    {
        C->data[k++] = A.data[i++];
    }

    while (j < B.length)
    {
        C->data[k++] = B.data[j++];
    }

    C->length = k;
}
```

时间复杂度：`O(m + n)`。

空间复杂度：`O(m + n)`，其中 `m` 和 `n` 分别是两个顺序表的长度。

### 6.2 删除顺序表中所有值为 x 的元素

方法一：边扫描边移动。

```c
void DeleteAllX(SeqList *L, ElemType x)
{
    int k = 0;

    for (int i = 0; i < L->length; i++)
    {
        if (L->data[i] != x)
        {
            L->data[k++] = L->data[i];
        }
    }

    L->length = k;
}
```

思路：

- `i` 用来扫描原顺序表。
- `k` 表示新顺序表当前应该写入的位置。
- 遇到不等于 `x` 的元素就保留。
- 遇到等于 `x` 的元素就跳过。

时间复杂度：`O(n)`。

空间复杂度：`O(1)`。

### 6.3 删除有序顺序表中的重复元素

题目：

删除递增有序顺序表中的重复元素，使每个元素只保留一个。

```c
void DeleteSame(SeqList *L)
{
    if (L->length == 0)
    {
        return;
    }

    int k = 0;

    for (int i = 1; i < L->length; i++)
    {
        if (L->data[i] != L->data[k])
        {
            L->data[++k] = L->data[i];
        }
    }

    L->length = k + 1;
}
```

思路：

- `k` 指向当前已保留区域的最后一个元素。
- `i` 扫描后面的元素。
- 如果 `data[i]` 和 `data[k]` 不同，说明遇到了新元素，将其保留。

时间复杂度：`O(n)`。

空间复杂度：`O(1)`。

### 6.4 顺序表逆置

题目：

将顺序表中的元素原地逆置。

```c
void Reverse(SeqList *L)
{
    int left = 0;
    int right = L->length - 1;

    while (left < right)
    {
        ElemType temp = L->data[left];
        L->data[left] = L->data[right];
        L->data[right] = temp;

        left++;
        right--;
    }
}
```

时间复杂度：`O(n)`。

空间复杂度：`O(1)`。

### 6.5 将奇数移动到偶数前面

题目：

调整顺序表，使所有奇数位于偶数之前。

思路：

- 使用双指针。
- `left` 从左向右找偶数。
- `right` 从右向左找奇数。
- 找到后交换二者。

代码：

```c
void Swap(ElemType *a, ElemType *b)
{
    ElemType temp = *a;
    *a = *b;
    *b = temp;
}

void MoveOddBeforeEven(SeqList *L)
{
    int left = 0;
    int right = L->length - 1;

    while (left < right)
    {
        while (left < right && L->data[left] % 2 != 0)
        {
            left++;
        }

        while (left < right && L->data[right] % 2 == 0)
        {
            right--;
        }

        if (left < right)
        {
            Swap(&L->data[left], &L->data[right]);
        }
    }
}
```

时间复杂度：`O(n)`。

空间复杂度：`O(1)`。

## 7. 顺序表操作复杂度总结

| 操作 | 时间复杂度 | 说明 |
| --- | --- | --- |
| 初始化 | `O(1)` | 设置表长为 0 |
| 求表长 | `O(1)` | 直接读取 `length` |
| 判断空表 | `O(1)` | 判断 `length == 0` |
| 按位查找 | `O(1)` | 顺序表支持随机访问 |
| 按值查找 | `O(n)` | 需要从头扫描 |
| 表尾插入 | `O(1)` | 不需要移动元素 |
| 表头插入 | `O(n)` | 所有元素后移 |
| 平均插入 | `O(n)` | 平均移动约一半元素 |
| 表尾删除 | `O(1)` | 不需要移动元素 |
| 表头删除 | `O(n)` | 所有元素前移 |
| 平均删除 | `O(n)` | 平均移动约一半元素 |
| 遍历 | `O(n)` | 访问所有元素 |

## 8. 易错点

1. 位序和下标混淆。

   位序从 `1` 开始，下标从 `0` 开始。第 `i` 个元素对应 `data[i - 1]`。

2. 插入位置范围写错。

   插入位置可以是 `1` 到 `length + 1`，因为允许在表尾后面插入。

3. 删除位置范围写错。

   删除位置只能是 `1` 到 `length`。

4. 插入移动方向写错。

   插入时应从后往前移动，避免数据被覆盖。

5. 删除移动方向写错。

   删除时应从前往后移动，用后面的元素覆盖前面的空位。

6. 忘记修改 `length`。

   插入成功后 `length++`，删除成功后 `length--`。

7. 顺序表满时仍然插入。

   插入前必须判断 `length >= MAXSIZE`。

8. 空表时进行非法访问。

   对空表进行删除、按位查找等操作前必须先判断合法性。

## 9. 复习口诀

```text
顺序存储地址连，随机访问最方便。
插入删除要搬家，查值还得逐个看。
位序一来下标零，i 位元素 i 减一。
插入后移从后起，删除前移从前行。
```

## 10. 最小完整示例

```c
#include <stdio.h>

#define MAXSIZE 100

typedef int ElemType;

typedef struct
{
    ElemType data[MAXSIZE];
    int length;
} SeqList;

void InitList(SeqList *L)
{
    L->length = 0;
}

int ListInsert(SeqList *L, int i, ElemType e)
{
    if (i < 1 || i > L->length + 1)
    {
        return 0;
    }

    if (L->length >= MAXSIZE)
    {
        return 0;
    }

    for (int j = L->length; j >= i; j--)
    {
        L->data[j] = L->data[j - 1];
    }

    L->data[i - 1] = e;
    L->length++;
    return 1;
}

int ListDelete(SeqList *L, int i, ElemType *e)
{
    if (i < 1 || i > L->length)
    {
        return 0;
    }

    *e = L->data[i - 1];

    for (int j = i; j < L->length; j++)
    {
        L->data[j - 1] = L->data[j];
    }

    L->length--;
    return 1;
}

void PrintList(SeqList L)
{
    for (int i = 0; i < L.length; i++)
    {
        printf("%d ", L.data[i]);
    }
    printf("\n");
}

int main(void)
{
    SeqList L;
    ElemType deleted;

    InitList(&L);

    ListInsert(&L, 1, 10);
    ListInsert(&L, 2, 20);
    ListInsert(&L, 3, 30);
    ListInsert(&L, 2, 15);

    printf("插入后：");
    PrintList(L);

    if (ListDelete(&L, 3, &deleted))
    {
        printf("删除的元素：%d\n", deleted);
    }

    printf("删除后：");
    PrintList(L);

    return 0;
}
```
