# 链表

## 1. 链表是什么

链表是一种线性数据结构。它和数组、顺序表一样，都用于存储一组具有线性关系的数据元素。

不同点在于：顺序表使用一段连续空间存储元素，而链表中的节点不要求连续存储。链表通过指针把一个个节点连接起来。

单链表节点一般由两部分组成：

1. 数据域：保存当前节点的数据。
2. 指针域：保存下一个节点的位置。

```c
typedef int ElemType;

typedef struct LNode
{
    ElemType data;
    struct LNode *next;
} LNode, *LinkList;
```

普通单链表示意：

```text
head
 |
 v
[1 | next] -> [2 | next] -> [3 | next] -> NULL
```

其中：

- `head` 指向第一个节点。
- 每个节点通过 `next` 指向后继节点。
- 最后一个节点的 `next` 指向 `NULL`。

## 2. 链表和顺序表的区别

| 对比项 | 顺序表 | 链表 |
|---|---|---|
| 存储方式 | 连续存储 | 不一定连续 |
| 随机访问 | 支持，`O(1)` | 不支持，`O(n)` |
| 按值查找 | `O(n)` | `O(n)` |
| 插入删除 | 需要移动元素 | 主要修改指针 |
| 空间开销 | 存储密度高 | 需要额外指针域 |
| 扩容 | 可能需要整体搬移 | 动态申请节点即可 |
| 缓存友好 | 较好 | 较差 |

简单理解：

- 顺序表适合频繁按下标访问。
- 链表适合频繁插入、删除，尤其是能快速找到操作位置时。

## 3. 链表的基本特点

优点：

- 不要求连续内存空间。
- 插入和删除时不需要大量移动元素。
- 容量可以按需动态增长。

缺点：

- 不支持随机访问。
- 查找第 `i` 个节点需要从头遍历。
- 每个节点需要额外存储指针。
- 指针操作容易出错。

## 4. 常见链表类型

### 4.1 单链表

每个节点只有一个 `next` 指针，只能从前往后遍历。

```text
1 -> 2 -> 3 -> NULL
```

特点：

- 结构简单。
- 只能单向移动。
- 删除某个节点时通常需要知道它的前驱节点。

### 4.2 双链表

每个节点有两个指针：

- `prior` 指向前驱节点。
- `next` 指向后继节点。

```c
typedef struct DNode
{
    ElemType data;
    struct DNode *prior;
    struct DNode *next;
} DNode, *DLinkList;
```

```text
NULL <- 1 <-> 2 <-> 3 -> NULL
```

特点：

- 可以双向遍历。
- 已知当前节点时，删除当前节点更方便。
- 每个节点多一个前驱指针，空间开销更大。

### 4.3 循环链表

循环链表的尾节点不指向 `NULL`，而是指向链表中的某个节点。

单循环链表通常让尾节点指向头节点：

```text
1 -> 2 -> 3
^         |
|_________|
```

特点：

- 从任意节点出发都可以继续访问后续节点。
- 遍历时不能用 `NULL` 判断结束。
- 常用于约瑟夫环等问题。

### 4.4 带头节点链表

带头节点链表会额外设置一个不存有效数据的头节点。

```text
head -> [头节点] -> [1] -> [2] -> [3] -> NULL
```

注意：

- `head` 指向头节点。
- `head->next` 才是第一个有效数据节点。

优点：

- 插入和删除第一个数据节点时，不需要特殊处理。
- 代码更统一。

### 4.5 不带头节点链表

不带头节点链表中，头指针直接指向第一个数据节点。

```text
head -> [1] -> [2] -> [3] -> NULL
```

如果链表为空：

```text
head == NULL
```

缺点：

- 删除或插入第一个节点时，可能需要单独处理头指针。

## 5. 头指针、头节点、第一个节点

这三个概念很容易混淆。

| 概念 | 含义 |
|---|---|
| 头指针 | 指向链表第一个节点的指针 |
| 头节点 | 链表开头额外设置的辅助节点，不一定存有效数据 |
| 第一个节点 | 第一个存放有效数据的节点 |

带头节点：

```text
head -> 头节点 -> 第一个数据节点 -> ...
```

不带头节点：

```text
head -> 第一个数据节点 -> ...
```

算法题里常见的 `dummy` 节点，其实就是临时创建的虚拟头节点。

## 6. 链表的复杂度

| 操作 | 单链表复杂度 | 说明 |
|---|---:|---|
| 求长度 | `O(n)` | 需要遍历 |
| 按位查找 | `O(n)` | 从头走到第 `i` 个节点 |
| 按值查找 | `O(n)` | 顺序扫描 |
| 头插 | `O(1)` | 修改头部指针 |
| 尾插 | `O(n)` 或 `O(1)` | 有尾指针时为 `O(1)` |
| 已知前驱插入 | `O(1)` | 修改两个指针 |
| 已知前驱删除后继 | `O(1)` | 修改一个指针 |
| 删除指定值 | `O(n)` | 需要查找 |

链表的优势不是查找快，而是插入删除时不需要移动大量元素。

## 7. 单链表基本操作

以下代码默认使用带头节点的单链表。

### 7.1 初始化

```c
int InitList(LinkList *L)
{
    *L = (LNode *)malloc(sizeof(LNode));
    if (*L == NULL)
    {
        return 0;
    }

    (*L)->next = NULL;
    return 1;
}
```

初始化后：

```text
head -> 头节点 -> NULL
```

### 7.2 判断空表

```c
int Empty(LinkList L)
{
    return L->next == NULL;
}
```

带头节点链表为空时，头节点仍然存在，只是 `head->next == NULL`。

### 7.3 求链表长度

```c
int Length(LinkList L)
{
    int len = 0;
    LNode *p = L->next;

    while (p != NULL)
    {
        len++;
        p = p->next;
    }

    return len;
}
```

时间复杂度：`O(n)`。

### 7.4 遍历链表

```c
void PrintList(LinkList L)
{
    LNode *p = L->next;

    while (p != NULL)
    {
        printf("%d ", p->data);
        p = p->next;
    }

    printf("\n");
}
```

常见错误：

```c
while (p->next != NULL)
```

这种写法会漏掉最后一个节点。如果要访问所有节点，应写：

```c
while (p != NULL)
```

### 7.5 按位查找

查找第 `i` 个数据节点。

```c
LNode *GetElem(LinkList L, int i)
{
    if (i < 1)
    {
        return NULL;
    }

    int j = 1;
    LNode *p = L->next;

    while (p != NULL && j < i)
    {
        p = p->next;
        j++;
    }

    return p;
}
```

说明：

- 如果找到，返回第 `i` 个节点。
- 如果不存在，返回 `NULL`。

时间复杂度：`O(n)`。

### 7.6 按值查找

```c
LNode *LocateElem(LinkList L, ElemType e)
{
    LNode *p = L->next;

    while (p != NULL && p->data != e)
    {
        p = p->next;
    }

    return p;
}
```

时间复杂度：`O(n)`。

### 7.7 头插法建立链表

头插法每次把新节点插到头节点后面。

```c
void CreateListHead(LinkList L, ElemType a[], int n)
{
    for (int i = 0; i < n; i++)
    {
        LNode *s = (LNode *)malloc(sizeof(LNode));
        s->data = a[i];

        s->next = L->next;
        L->next = s;
    }
}
```

如果数组是：

```text
1 2 3 4
```

头插法建立后的链表是：

```text
4 -> 3 -> 2 -> 1
```

特点：

- 插入简单，时间复杂度 `O(n)`。
- 得到的链表顺序与输入顺序相反。

### 7.8 尾插法建立链表

尾插法每次把新节点插到链表尾部。

```c
void CreateListTail(LinkList L, ElemType a[], int n)
{
    LNode *tail = L;

    for (int i = 0; i < n; i++)
    {
        LNode *s = (LNode *)malloc(sizeof(LNode));
        s->data = a[i];
        s->next = NULL;

        tail->next = s;
        tail = s;
    }
}
```

如果数组是：

```text
1 2 3 4
```

尾插法建立后的链表是：

```text
1 -> 2 -> 3 -> 4
```

特点：

- 保持输入顺序。
- 需要维护尾指针。

### 7.9 插入节点

在第 `i` 个位置插入元素 `e`。

```c
int ListInsert(LinkList L, int i, ElemType e)
{
    if (i < 1)
    {
        return 0;
    }

    LNode *p = L;
    int j = 0;

    while (p != NULL && j < i - 1)
    {
        p = p->next;
        j++;
    }

    if (p == NULL)
    {
        return 0;
    }

    LNode *s = (LNode *)malloc(sizeof(LNode));
    if (s == NULL)
    {
        return 0;
    }

    s->data = e;
    s->next = p->next;
    p->next = s;

    return 1;
}
```

关键点：

- 插入第 `i` 个位置，需要先找到第 `i - 1` 个节点。
- 插入第 `1` 个位置时，前驱节点是头节点。
- 修改指针顺序不能写反。

正确顺序：

```c
s->next = p->next;
p->next = s;
```

### 7.10 删除节点

删除第 `i` 个数据节点，并用 `e` 返回被删除的值。

```c
int ListDelete(LinkList L, int i, ElemType *e)
{
    if (i < 1)
    {
        return 0;
    }

    LNode *p = L;
    int j = 0;

    while (p != NULL && j < i - 1)
    {
        p = p->next;
        j++;
    }

    if (p == NULL || p->next == NULL)
    {
        return 0;
    }

    LNode *q = p->next;
    *e = q->data;
    p->next = q->next;
    free(q);

    return 1;
}
```

关键点：

- 删除第 `i` 个节点，需要找到第 `i - 1` 个节点。
- 被删除节点是 `p->next`。
- 释放节点前要先保存它。

### 7.11 销毁链表

```c
void DestroyList(LinkList *L)
{
    LNode *p = *L;

    while (p != NULL)
    {
        LNode *next = p->next;
        free(p);
        p = next;
    }

    *L = NULL;
}
```

销毁时必须先保存 `next`，否则释放当前节点后就无法访问后继节点。

## 8. 不带头节点的操作特点

不带头节点时，头指针直接指向第一个数据节点。

插入到表头：

```c
LNode *s = (LNode *)malloc(sizeof(LNode));
s->data = e;
s->next = head;
head = s;
```

删除表头：

```c
if (head != NULL)
{
    LNode *q = head;
    head = head->next;
    free(q);
}
```

不带头节点的麻烦在于：

- 插入第一个位置会改变 `head`。
- 删除第一个节点也会改变 `head`。
- 代码里经常需要单独处理头节点。

所以教材和算法题中经常使用头节点或虚拟头节点来统一操作。

## 9. 虚拟头节点 dummy

虚拟头节点是链表题中非常重要的技巧。

在 C++ 算法题中常写：

```cpp
ListNode dummy(0);
dummy.next = head;
ListNode *cur = &dummy;
```

最后返回：

```cpp
return dummy.next;
```

适用场景：

- 删除节点时可能删除头节点。
- 合并两个链表。
- 分割链表。
- 两两交换节点。
- 反转部分链表。

核心作用：让原来的头节点也有一个前驱节点。

## 10. 链表题常用技巧

### 10.1 保存后继节点

只要接下来要修改 `cur->next`，就要考虑是否先保存原来的后继节点。

```c
LNode *next = cur->next;
cur->next = prev;
prev = cur;
cur = next;
```

如果不保存 `next`，原链表后半段可能丢失。

### 10.2 找前驱节点

很多链表删除和插入都不是直接操作目标节点，而是先找目标节点的前驱。

例如删除 `cur->next`：

```c
LNode *q = cur->next;
cur->next = q->next;
free(q);
```

### 10.3 快慢指针

快慢指针常用于找中点、判环、找环入口。

```cpp
ListNode *slow = head;
ListNode *fast = head;

while (fast != nullptr && fast->next != nullptr)
{
    slow = slow->next;
    fast = fast->next->next;
}
```

### 10.4 前后指针

让两个指针保持固定距离，常用于找倒数第 `k` 个节点。

```cpp
ListNode *fast = head;
ListNode *slow = head;

for (int i = 0; i < k; i++)
{
    fast = fast->next;
}

while (fast != nullptr)
{
    fast = fast->next;
    slow = slow->next;
}
```

## 11. 反转链表

### 11.1 反转整个链表

```cpp
ListNode *reverseList(ListNode *head)
{
    ListNode *prev = nullptr;
    ListNode *cur = head;

    while (cur != nullptr)
    {
        ListNode *next = cur->next;
        cur->next = prev;
        prev = cur;
        cur = next;
    }

    return prev;
}
```

过程：

```text
原链表：
1 -> 2 -> 3 -> NULL

反转后：
3 -> 2 -> 1 -> NULL
```

复杂度：

- 时间复杂度：`O(n)`。
- 空间复杂度：`O(1)`。

### 11.2 递归反转链表

```cpp
ListNode *reverseList(ListNode *head)
{
    if (head == nullptr || head->next == nullptr)
    {
        return head;
    }

    ListNode *newHead = reverseList(head->next);
    head->next->next = head;
    head->next = nullptr;

    return newHead;
}
```

递归写法简洁，但链表很长时可能栈溢出。

### 11.3 反转区间

反转链表中 `[left, right]` 区间。

```cpp
ListNode *reverseBetween(ListNode *head, int left, int right)
{
    if (head == nullptr || left == right)
    {
        return head;
    }

    ListNode dummy(0);
    dummy.next = head;

    ListNode *pre = &dummy;
    for (int i = 1; i < left; i++)
    {
        pre = pre->next;
    }

    ListNode *cur = pre->next;

    for (int i = 0; i < right - left; i++)
    {
        ListNode *next = cur->next;
        cur->next = next->next;
        next->next = pre->next;
        pre->next = next;
    }

    return dummy.next;
}
```

这段代码使用头插法反转区间。

### 11.4 K 个一组反转

```cpp
ListNode *reverseKGroup(ListNode *head, int k)
{
    ListNode dummy(0);
    dummy.next = head;

    ListNode *groupPrev = &dummy;

    while (true)
    {
        ListNode *kth = groupPrev;
        for (int i = 0; i < k && kth != nullptr; i++)
        {
            kth = kth->next;
        }

        if (kth == nullptr)
        {
            break;
        }

        ListNode *groupNext = kth->next;
        ListNode *prev = groupNext;
        ListNode *cur = groupPrev->next;

        while (cur != groupNext)
        {
            ListNode *next = cur->next;
            cur->next = prev;
            prev = cur;
            cur = next;
        }

        ListNode *oldHead = groupPrev->next;
        groupPrev->next = kth;
        groupPrev = oldHead;
    }

    return dummy.next;
}
```

## 12. 快慢指针经典问题

### 12.1 找链表中点

```cpp
ListNode *middleNode(ListNode *head)
{
    ListNode *slow = head;
    ListNode *fast = head;

    while (fast != nullptr && fast->next != nullptr)
    {
        slow = slow->next;
        fast = fast->next->next;
    }

    return slow;
}
```

如果链表长度为偶数，该写法返回第二个中间节点。

### 12.2 判断链表是否有环

```cpp
bool hasCycle(ListNode *head)
{
    ListNode *slow = head;
    ListNode *fast = head;

    while (fast != nullptr && fast->next != nullptr)
    {
        slow = slow->next;
        fast = fast->next->next;

        if (slow == fast)
        {
            return true;
        }
    }

    return false;
}
```

### 12.3 找环入口

```cpp
ListNode *detectCycle(ListNode *head)
{
    ListNode *slow = head;
    ListNode *fast = head;

    while (fast != nullptr && fast->next != nullptr)
    {
        slow = slow->next;
        fast = fast->next->next;

        if (slow == fast)
        {
            ListNode *p = head;
            ListNode *q = slow;

            while (p != q)
            {
                p = p->next;
                q = q->next;
            }

            return p;
        }
    }

    return nullptr;
}
```

记忆：

- 第一次相遇：证明有环。
- 第二次相遇：找到入口。

## 13. 双指针经典问题

### 13.1 删除倒数第 n 个节点

```cpp
ListNode *removeNthFromEnd(ListNode *head, int n)
{
    ListNode dummy(0);
    dummy.next = head;

    ListNode *fast = &dummy;
    ListNode *slow = &dummy;

    for (int i = 0; i < n + 1; i++)
    {
        fast = fast->next;
    }

    while (fast != nullptr)
    {
        fast = fast->next;
        slow = slow->next;
    }

    ListNode *del = slow->next;
    slow->next = del->next;
    delete del;

    return dummy.next;
}
```

### 13.2 找倒数第 k 个节点

```cpp
ListNode *kthFromEnd(ListNode *head, int k)
{
    ListNode *fast = head;
    ListNode *slow = head;

    for (int i = 0; i < k; i++)
    {
        if (fast == nullptr)
        {
            return nullptr;
        }
        fast = fast->next;
    }

    while (fast != nullptr)
    {
        fast = fast->next;
        slow = slow->next;
    }

    return slow;
}
```

### 13.3 两个链表相交

```cpp
ListNode *getIntersectionNode(ListNode *headA, ListNode *headB)
{
    ListNode *a = headA;
    ListNode *b = headB;

    while (a != b)
    {
        a = (a == nullptr) ? headB : a->next;
        b = (b == nullptr) ? headA : b->next;
    }

    return a;
}
```

如果相交，两个指针会在交点相遇；如果不相交，最终都会变成 `nullptr`。

## 14. 合并和构造链表

### 14.1 合并两个有序链表

```cpp
ListNode *mergeTwoLists(ListNode *l1, ListNode *l2)
{
    ListNode dummy(0);
    ListNode *tail = &dummy;

    while (l1 != nullptr && l2 != nullptr)
    {
        if (l1->val <= l2->val)
        {
            tail->next = l1;
            l1 = l1->next;
        }
        else
        {
            tail->next = l2;
            l2 = l2->next;
        }

        tail = tail->next;
    }

    tail->next = (l1 != nullptr) ? l1 : l2;
    return dummy.next;
}
```

### 14.2 合并 K 个有序链表

常见方法：

1. 使用小根堆，时间复杂度 `O(N log k)`。
2. 分治合并，时间复杂度 `O(N log k)`。

其中 `N` 是总节点数，`k` 是链表个数。

分治思路：

```cpp
ListNode *mergeRange(vector<ListNode *> &lists, int left, int right)
{
    if (left > right)
    {
        return nullptr;
    }

    if (left == right)
    {
        return lists[left];
    }

    int mid = left + (right - left) / 2;
    ListNode *l1 = mergeRange(lists, left, mid);
    ListNode *l2 = mergeRange(lists, mid + 1, right);

    return mergeTwoLists(l1, l2);
}
```

## 15. 链表排序

链表排序常用归并排序。

原因：

- 链表不适合随机访问。
- 归并排序只需要拆分和合并链表。

```cpp
ListNode *sortList(ListNode *head)
{
    if (head == nullptr || head->next == nullptr)
    {
        return head;
    }

    ListNode *slow = head;
    ListNode *fast = head->next;

    while (fast != nullptr && fast->next != nullptr)
    {
        slow = slow->next;
        fast = fast->next->next;
    }

    ListNode *second = slow->next;
    slow->next = nullptr;

    ListNode *left = sortList(head);
    ListNode *right = sortList(second);

    return mergeTwoLists(left, right);
}
```

复杂度：

- 时间复杂度：`O(n log n)`。
- 递归空间复杂度：`O(log n)`。

## 16. 回文链表

判断链表是否是回文。

思路：

1. 快慢指针找中点。
2. 反转后半部分。
3. 前后两段逐个比较。
4. 可选：恢复链表。

```cpp
bool isPalindrome(ListNode *head)
{
    if (head == nullptr || head->next == nullptr)
    {
        return true;
    }

    ListNode *slow = head;
    ListNode *fast = head;

    while (fast != nullptr && fast->next != nullptr)
    {
        slow = slow->next;
        fast = fast->next->next;
    }

    ListNode *second = reverseList(slow);
    ListNode *p1 = head;
    ListNode *p2 = second;

    bool ok = true;
    while (p2 != nullptr)
    {
        if (p1->val != p2->val)
        {
            ok = false;
            break;
        }

        p1 = p1->next;
        p2 = p2->next;
    }

    reverseList(second);
    return ok;
}
```

## 17. 重排链表

题目形式：

```text
L0 -> L1 -> L2 -> ... -> Ln
```

重排为：

```text
L0 -> Ln -> L1 -> Ln-1 -> L2 -> Ln-2 -> ...
```

思路：

1. 找中点。
2. 断开成前后两段。
3. 反转后半段。
4. 交替合并。

```cpp
void reorderList(ListNode *head)
{
    if (head == nullptr || head->next == nullptr)
    {
        return;
    }

    ListNode *slow = head;
    ListNode *fast = head;

    while (fast->next != nullptr && fast->next->next != nullptr)
    {
        slow = slow->next;
        fast = fast->next->next;
    }

    ListNode *second = slow->next;
    slow->next = nullptr;
    second = reverseList(second);

    ListNode *first = head;

    while (second != nullptr)
    {
        ListNode *next1 = first->next;
        ListNode *next2 = second->next;

        first->next = second;
        second->next = next1;

        first = next1;
        second = next2;
    }
}
```

## 18. 两数相加

链表逆序存储数字，每个节点保存一位。

```text
2 -> 4 -> 3 表示 342
5 -> 6 -> 4 表示 465

结果：
7 -> 0 -> 8
```

代码：

```cpp
ListNode *addTwoNumbers(ListNode *l1, ListNode *l2)
{
    ListNode dummy(0);
    ListNode *tail = &dummy;
    int carry = 0;

    while (l1 != nullptr || l2 != nullptr || carry != 0)
    {
        int x = (l1 != nullptr) ? l1->val : 0;
        int y = (l2 != nullptr) ? l2->val : 0;

        int sum = x + y + carry;
        carry = sum / 10;

        tail->next = new ListNode(sum % 10);
        tail = tail->next;

        if (l1 != nullptr)
        {
            l1 = l1->next;
        }

        if (l2 != nullptr)
        {
            l2 = l2->next;
        }
    }

    return dummy.next;
}
```

## 19. 分割链表

给定 `x`，把小于 `x` 的节点放前面，大于等于 `x` 的节点放后面，并保持相对顺序。

```cpp
ListNode *partition(ListNode *head, int x)
{
    ListNode smallDummy(0);
    ListNode largeDummy(0);

    ListNode *small = &smallDummy;
    ListNode *large = &largeDummy;

    ListNode *cur = head;

    while (cur != nullptr)
    {
        ListNode *next = cur->next;
        cur->next = nullptr;

        if (cur->val < x)
        {
            small->next = cur;
            small = small->next;
        }
        else
        {
            large->next = cur;
            large = large->next;
        }

        cur = next;
    }

    small->next = largeDummy.next;
    return smallDummy.next;
}
```

关键点：每次先保存 `next`，并断开 `cur->next`。

## 20. 两两交换节点

ListNode *swapPairs(ListNode *head)
{
    ListNode dummy(0);
    dummy.next = head;

    ListNode *cur = &dummy;

    while (cur->next != nullptr && cur->next->next != nullptr)
    {
        ListNode *first = cur->next;

        second->next = first;

## 21. 删除重复节点

### 21.1 有序链表保留一个重复值

```cpp
ListNode *deleteDuplicates(ListNode *head)
{
    ListNode *cur = head;

    while (cur != nullptr && cur->next != nullptr)
    {
        if (cur->val == cur->next->val)
        {
            ListNode *del = cur->next;
            cur->next = del->next;
            delete del;
        }
        else
        {
            cur = cur->next;
        }
    }

    return head;
}
```

### 21.2 有序链表删除所有重复值

```cpp
ListNode *deleteDuplicatesAll(ListNode *head)
{
    ListNode dummy(0);
    dummy.next = head;

    ListNode *pre = &dummy;

    while (pre->next != nullptr && pre->next->next != nullptr)
    {
        if (pre->next->val == pre->next->next->val)
        {
            int x = pre->next->val;

            while (pre->next != nullptr && pre->next->val == x)
            {
                ListNode *del = pre->next;
                pre->next = del->next;
                delete del;
            }
        }
        else
        {
            pre = pre->next;
        }
    }

    return dummy.next;
}
```

## 22. 双链表操作

### 22.1 在节点后插入

```c
void InsertAfter(DNode *p, DNode *s)
{
    s->next = p->next;
    s->prior = p;

    if (p->next != NULL)
    {
        p->next->prior = s;
    }

    p->next = s;
}
```

### 22.2 删除当前节点

```c
void DeleteNode(DNode *p)
{
    if (p->prior != NULL)
    {
        p->prior->next = p->next;
    }

    if (p->next != NULL)
    {
        p->next->prior = p->prior;
    }

    free(p);
}
```

双链表适用于：

- LRU 缓存。
- 浏览器前进后退。
- 需要从中间快速删除节点的场景。

## 23. LRU 缓存与双链表

LRU 是 Least Recently Used，表示最近最少使用。

常见设计：

- 哈希表：根据 key 快速找到节点。
- 双链表：维护使用顺序。

约定：

- 越靠近头部表示越新。
- 越靠近尾部表示越旧。

核心操作：

1. `get(key)`：如果存在，把节点移动到头部。
2. `put(key, value)`：如果存在，更新并移动到头部。
3. 容量超限时，删除尾部最旧节点。

为什么用双链表：

- 已知节点时，可以 `O(1)` 删除当前节点。
- 单链表删除当前节点还需要找前驱节点。

## 24. 链表调试方法

### 24.1 画节点而不是只画值

不要只写：

```text
1 -> 2 -> 3
```

更推荐写：

```text
A(1) -> B(2) -> C(3)
```

因为链表题关心的是节点本身，不只是节点值。两个值相等的节点不一定是同一个节点。

### 24.2 明确每个指针的含义

例如：

```c
LNode *pre = L;
```

要明确 `pre` 表示待操作节点的前驱，而不是随便遍历的当前节点。

### 24.3 修改指针前保存后继

```c
LNode *next = cur->next;
```

只要接下来要改变 `cur->next`，就先判断是否需要保存原后继。

## 25. 常见错误

### 25.1 修改指针顺序错误

插入节点时正确顺序：

```c
s->next = p->next;
p->next = s;
```

如果先写：

```c
p->next = s;
s->next = p->next;
```

会导致 `s->next` 指向自己，原后半段链表丢失。

### 25.2 忘记处理头节点

删除第一个节点或在第一个位置插入时，头指针可能改变。

解决办法：

- 使用带头节点链表。
- 或在算法题中使用 `dummy`。

### 25.3 遍历条件写错

访问 `fast->next` 前必须保证 `fast != NULL`。

正确：

```cpp
while (fast != nullptr && fast->next != nullptr)
```

错误：

```cpp
while (fast->next != nullptr && fast != nullptr)
```

### 25.4 删除后指针移动错误

删除当前后继节点后，当前指针不一定移动。

```c
if (cur->next->data == x)
{
    LNode *q = cur->next;
    cur->next = q->next;
    free(q);
}
else
{
    cur = cur->next;
}
```

如果删除后仍然移动，可能跳过连续的目标节点。

### 25.5 忘记断开链表

拆分、分割、重排链表时，经常需要断开旧连接。

```cpp
slow->next = nullptr;
```

或：

```cpp
cur->next = nullptr;
```

否则可能形成环或错误连接。

### 25.6 释放节点后继续访问

错误：

```c
free(p);
p = p->next;
```

正确：

```c
LNode *next = p->next;
free(p);
p = next;
```

## 28. 总结

链表题的核心是指针关系。

最重要的三件事：

1. 找到要操作节点的前驱节点。
2. 修改指针前保存必要的后继节点。
3. 重新连接链表时保证头尾都正确。

最常用的四个工具：

1. `dummy`：统一头节点边界。
2. `prev / cur / next`：反转链表。
3. `slow / fast`：找中点、判环、找入口。
4. `tail`：构造新链表或合并链表。

写链表代码前，最好先画出节点和指针，明确当前指针、前驱节点、后继节点分别是谁。只要指针关系清楚，大部分链表题都可以拆成几步稳定操作。
