# 顺序表

## 1. 顺序表是什么

顺序表是线性表的顺序存储结构。它使用一段地址连续的存储空间，按照元素的逻辑顺序依次存放数据。

线性表的逻辑形式为：

```text
L = (a1, a2, ..., ai, ..., an)
```

顺序表在内存中的存储形式可以理解为：

```text
下标：  0    1    2    3        n - 1
元素： a1   a2   a3   a4  ...   an
```

也就是说：

- 逻辑上相邻的元素，物理存储位置也相邻。
- 顺序表可以通过下标直接访问任意元素。
- 顺序表本质上通常由数组实现。

## 2. 线性表与顺序表的关系

线性表是一种逻辑结构，它描述元素之间的关系。

顺序表是一种存储结构，它描述线性表在内存中如何存放。

线性表主要有两种常见存储方式：

| 存储方式 | 名称 | 特点 |
|---|---|---|
| 顺序存储 | 顺序表 | 地址连续，支持随机访问 |
| 链式存储 | 链表 | 地址不一定连续，通过指针连接 |

所以：

```text
线性表 = 逻辑结构
顺序表 = 线性表的顺序存储实现
链表   = 线性表的链式存储实现
```

## 3. 位序与下标

学习顺序表时，一定要区分“位序”和“数组下标”。

| 概念 | 起始值 | 例子 |
|---|---:|---|
| 位序 | `1` | 第 1 个元素、第 2 个元素 |
| 下标 | `0` | `data[0]`、`data[1]` |

如果元素的位序为 `i`，那么它在 C 语言数组中的下标是：

```text
i - 1
```

对应关系：

```text
第 1 个元素 -> data[0]
第 2 个元素 -> data[1]
第 i 个元素 -> data[i - 1]
第 n 个元素 -> data[n - 1]
```

这是顺序表最容易出错的地方之一。

## 4. 顺序表的地址计算

顺序表中的元素连续存储，因此可以直接计算任意元素的地址。

假设：

- 第一个元素 `a1` 的地址为 `LOC(a1)`。
- 每个元素占用 `sizeof(ElemType)` 个字节。
- 要访问第 `i` 个元素 `ai`。

那么：

```text
LOC(ai) = LOC(a1) + (i - 1) * sizeof(ElemType)
```

如果使用数组下标 `k`：

```text
LOC(data[k]) = LOC(data[0]) + k * sizeof(ElemType)
```

顺序表之所以能 `O(1)` 按位查找，就是因为可以通过这个公式直接定位元素地址。

## 5. 顺序表的特点

### 5.1 优点

- 支持随机访问，可以通过下标直接访问任意元素。
- 按位查找时间复杂度为 `O(1)`。
- 存储密度高，只存数据元素，不需要额外指针域。
- 结构简单，遍历方便。
- 缓存友好，连续内存对 CPU cache 更友好。

### 5.2 缺点

- 插入和删除时通常需要移动大量元素。
- 静态顺序表容量固定，可能空间不足或空间浪费。
- 动态顺序表扩容时，需要重新申请空间并搬移元素。
- 如果频繁在表头或中间插入删除，效率不如链表。

## 6. 顺序表和链表的区别

| 对比项 | 顺序表 | 链表 |
|---|---|---|
| 存储空间 | 连续 | 不一定连续 |
| 随机访问 | 支持，`O(1)` | 不支持，`O(n)` |
| 按值查找 | `O(n)` | `O(n)` |
| 插入删除 | 需要移动元素，通常 `O(n)` | 已知前驱时改指针，`O(1)` |
| 空间开销 | 存储密度高 | 需要额外指针域 |
| 扩容 | 可能需要搬移元素 | 动态申请节点即可 |
| 缓存友好 | 较好 | 较差 |

简单选择：

- 经常按下标访问：优先顺序表。
- 经常遍历、排序、二分：优先顺序表。
- 经常在中间插入删除，并且能快速定位位置：可以考虑链表。

## 7. 顺序表的存储结构

### 7.1 静态顺序表

静态顺序表使用固定大小数组保存元素。

```c
#include <stdio.h>

#define MAXSIZE 100

typedef int ElemType;

typedef struct
{
    ElemType data[MAXSIZE];
    int length;
} SeqList;
```

含义：

- `data`：存放数据元素的数组。
- `length`：当前顺序表中的元素个数。
- `MAXSIZE`：顺序表最多能容纳的元素个数。

特点：

- 实现简单。
- 容量固定。
- 超过 `MAXSIZE` 后无法继续插入。

### 7.2 动态顺序表

动态顺序表使用动态申请的连续空间保存元素。

```c
#include <stdio.h>
#include <stdlib.h>

#define INIT_SIZE 10

typedef int ElemType;

typedef struct
{
    ElemType *data;
    int length;
    int capacity;
} SeqList;
```

含义：

- `data`：指向动态申请的连续存储空间。
- `length`：当前元素个数。
- `capacity`：当前容量。

特点：

- 容量可以扩展。
- 扩容时需要申请新空间并复制旧元素。
- C++ 中的 `vector` 可以理解为动态顺序表的一种典型实现。

## 8. 静态顺序表基本操作

### 8.1 初始化

```c
void InitList(SeqList *L)
{
    L->length = 0;
}
```

初始化只需要把当前长度置为 `0`。

时间复杂度：`O(1)`。

### 8.2 求表长

```c
int Length(SeqList L)
{
    return L.length;
}
```

时间复杂度：`O(1)`。

### 8.3 判断空表

```c
int IsEmpty(SeqList L)
{
    return L.length == 0;
}
```

时间复杂度：`O(1)`。

### 8.4 判断满表

```c
int IsFull(SeqList L)
{
    return L.length >= MAXSIZE;
}
```

时间复杂度：`O(1)`。

### 8.5 按位查找

按位查找是根据位序 `i` 查找第 `i` 个元素。

```c
int GetElem(SeqList L, int i, ElemType *e)
{
    if (i < 1 || i > L.length)
    {
        return 0;
    }

    *e = L.data[i - 1];
    return 1;
}
```

要点：

- 合法位序范围是 `[1, length]`。
- 位序 `i` 对应下标 `i - 1`。

时间复杂度：`O(1)`。

### 8.6 按值查找

按值查找是查找第一个值等于 `e` 的元素位置。

```c
int LocateElem(SeqList L, ElemType e)
{
    for (int i = 0; i < L.length; i++)
    {
        if (L.data[i] == e)
        {
            return i + 1;
        }
    }

    return 0;
}
```

返回值：

- 找到：返回元素位序。
- 未找到：返回 `0`。

时间复杂度：`O(n)`。

### 8.7 遍历输出

```c
void PrintList(SeqList L)
{
    for (int i = 0; i < L.length; i++)
    {
        printf("%d ", L.data[i]);
    }
    printf("\n");
}
```

时间复杂度：`O(n)`。

### 8.8 尾插

在顺序表末尾添加元素。

```c
int PushBack(SeqList *L, ElemType e)
{
    if (L->length >= MAXSIZE)
    {
        return 0;
    }

    L->data[L->length] = e;
    L->length++;
    return 1;
}
```

时间复杂度：`O(1)`。

### 8.9 指定位序插入

在第 `i` 个位置插入元素 `e`。

插入前：

```text
L = (a1, a2, ..., ai, ..., an)
```

插入后：

```text
L = (a1, a2, ..., e, ai, ..., an)
```

代码：

```c
int ListInsert(SeqList *L, int i, ElemType e)
{
    if (i < 1 || i > L->length + 1)
    {
        return 0;
    }

    if (L->length >= MAXSIZE)
    {
        return 0;
    }

    for (int j = L->length; j >= i; j--)
    {
        L->data[j] = L->data[j - 1];
    }

    L->data[i - 1] = e;
    L->length++;
    return 1;
}
```

关键点：

- 插入位置合法范围是 `[1, length + 1]`。
- `i == length + 1` 表示尾插。
- 移动元素时必须从后往前移动，避免覆盖数据。

移动示意：

```text
插入前：
下标： 0   1   2   3
元素：10  20  30  40

在位序 2 插入 99：
先移动 40，再移动 30，再移动 20。

插入后：
下标： 0   1   2   3   4
元素：10  99  20  30  40
```

时间复杂度：

- 最好情况：表尾插入，`O(1)`。
- 最坏情况：表头插入，`O(n)`。
- 平均情况：`O(n)`。

### 8.10 指定下标插入

如果使用数组下标 `pos`，合法范围是 `[0, length]`。

```c
int InsertAt(SeqList *L, int pos, ElemType e)
{
    if (pos < 0 || pos > L->length)
    {
        return 0;
    }

    if (L->length >= MAXSIZE)
    {
        return 0;
    }

    for (int j = L->length; j > pos; j--)
    {
        L->data[j] = L->data[j - 1];
    }

    L->data[pos] = e;
    L->length++;
    return 1;
}
```

注意：

- 位序插入用 `i`，范围 `[1, length + 1]`。
- 下标插入用 `pos`，范围 `[0, length]`。
- 不要把二者混在一起。

### 8.11 删除指定位序元素

删除第 `i` 个位置的元素，并用 `e` 返回被删除的值。

删除前：

```text
L = (a1, a2, ..., ai, ai+1, ..., an)
```

删除后：

```text
L = (a1, a2, ..., ai-1, ai+1, ..., an)
```

代码：

```c
int ListDelete(SeqList *L, int i, ElemType *e)
{
    if (i < 1 || i > L->length)
    {
        return 0;
    }

    *e = L->data[i - 1];

    for (int j = i; j < L->length; j++)
    {
        L->data[j - 1] = L->data[j];
    }

    L->length--;
    return 1;
}
```

关键点：

- 删除位置合法范围是 `[1, length]`。
- 删除第 `i` 个元素时，从第 `i + 1` 个元素开始向前移动。
- 删除完成后 `length--`。

时间复杂度：

- 最好情况：删除表尾，`O(1)`。
- 最坏情况：删除表头，`O(n)`。
- 平均情况：`O(n)`。

### 8.12 删除指定下标元素

如果使用下标 `pos`，合法范围是 `[0, length - 1]`。

```c
int DeleteAt(SeqList *L, int pos, ElemType *e)
{
    if (pos < 0 || pos >= L->length)
    {
        return 0;
    }

    *e = L->data[pos];

    for (int j = pos; j < L->length - 1; j++)
    {
        L->data[j] = L->data[j + 1];
    }

    L->length--;
    return 1;
}
```

注意：必须先判断 `pos` 合法，再访问 `L->data[pos]`。

### 8.13 修改元素

按位序修改：

```c
int SetElem(SeqList *L, int i, ElemType e)
{
    if (i < 1 || i > L->length)
    {
        return 0;
    }

    L->data[i - 1] = e;
    return 1;
}
```

时间复杂度：`O(1)`。

### 8.14 清空顺序表

```c
void ClearList(SeqList *L)
{
    L->length = 0;
}
```

对于静态顺序表，清空通常只需要把 `length` 置为 `0`，不需要逐个清除数组中的旧值。

## 9. 动态顺序表基本操作

### 9.1 初始化

```c
#include <stdlib.h>

#define INIT_SIZE 10

int InitList(SeqList *L)
{
    L->data = (ElemType *)malloc(sizeof(ElemType) * INIT_SIZE);
    if (L->data == NULL)
    {
        return 0;
    }

    L->length = 0;
    L->capacity = INIT_SIZE;
    return 1;
}
```

### 9.2 销毁

```c
void DestroyList(SeqList *L)
{
    free(L->data);
    L->data = NULL;
    L->length = 0;
    L->capacity = 0;
}
```

要点：

- `free` 之后最好把指针置为 `NULL`。
- 避免野指针和重复释放。

### 9.3 扩容

当 `length == capacity` 时，需要扩容。

```c
int ExpandList(SeqList *L)
{
    int newCapacity = L->capacity * 2;
    ElemType *newData = (ElemType *)realloc(L->data, sizeof(ElemType) * newCapacity);

    if (newData == NULL)
    {
        return 0;
    }

    L->data = newData;
    L->capacity = newCapacity;
    return 1;
}
```

说明：

- 常见扩容策略是容量翻倍。
- `realloc` 可能在原地扩容，也可能搬到新地址。
- 如果 `realloc` 失败，原指针仍然有效，因此要先用临时指针接收返回值。

### 9.4 动态尾插

```c
int PushBack(SeqList *L, ElemType e)
{
    if (L->length >= L->capacity)
    {
        if (!ExpandList(L))
        {
            return 0;
        }
    }

    L->data[L->length] = e;
    L->length++;
    return 1;
}
```

尾插的摊还时间复杂度是 `O(1)`。

为什么不是每次都 `O(1)`：

- 大多数尾插只写入一个元素，`O(1)`。
- 偶尔扩容时要搬移所有元素，`O(n)`。
- 采用倍增扩容后，平均到每次插入上，摊还复杂度为 `O(1)`。

### 9.5 动态插入

```c
int ListInsert(SeqList *L, int i, ElemType e)
{
    if (i < 1 || i > L->length + 1)
    {
        return 0;
    }

    if (L->length >= L->capacity)
    {
        if (!ExpandList(L))
        {
            return 0;
        }
    }

    for (int j = L->length; j >= i; j--)
    {
        L->data[j] = L->data[j - 1];
    }

    L->data[i - 1] = e;
    L->length++;
    return 1;
}
```

时间复杂度：

- 尾插且不扩容：`O(1)`。
- 头插或中间插入：`O(n)`。
- 扩容时还要额外搬移元素。

### 9.6 动态顺序表完整结构建议

如果写动态顺序表，建议结构体里同时保存 `length` 和 `capacity`。

不建议只保存 `length`，然后把最大容量写死成 `MAXSIZE`，否则动态顺序表就失去了动态扩容的意义。

```c
typedef struct
{
    ElemType *data;
    int length;
    int capacity;
} SeqList;
```

## 10. 操作复杂度总结

| 操作 | 时间复杂度 | 说明 |
|---|---:|---|
| 初始化 | `O(1)` | 设置长度和容量 |
| 求表长 | `O(1)` | 直接读取 `length` |
| 判断空表 | `O(1)` | 判断 `length == 0` |
| 判断满表 | `O(1)` | 静态表判断 `length == MAXSIZE` |
| 按位查找 | `O(1)` | 通过地址公式直接定位 |
| 按值查找 | `O(n)` | 需要顺序扫描 |
| 表尾插入 | `O(1)` | 静态表不满时直接写入 |
| 动态表尾插 | 摊还 `O(1)` | 扩容时单次可能 `O(n)` |
| 表头插入 | `O(n)` | 所有元素后移 |
| 中间插入 | `O(n)` | 平均移动约一半元素 |
| 表尾删除 | `O(1)` | 只需 `length--` |
| 表头删除 | `O(n)` | 所有元素前移 |
| 中间删除 | `O(n)` | 平均移动约一半元素 |
| 遍历 | `O(n)` | 访问所有元素 |
| 原地逆置 | `O(n)` | 双指针交换 |

## 11. 插入和删除的移动次数

### 11.1 插入移动次数

在长度为 `n` 的顺序表中，第 `i` 个位置插入元素。

需要移动的元素个数：

```text
n - i + 1
```

特殊情况：

- 在第 `1` 个位置插入：移动 `n` 个元素。
- 在第 `n + 1` 个位置插入：移动 `0` 个元素。

如果每个位置插入概率相同，平均移动次数为：

```text
n / 2
```

### 11.2 删除移动次数

在长度为 `n` 的顺序表中，删除第 `i` 个元素。

需要移动的元素个数：

```text
n - i
```

特殊情况：

- 删除第 `1` 个元素：移动 `n - 1` 个元素。
- 删除第 `n` 个元素：移动 `0` 个元素。

如果每个位置删除概率相同，平均移动次数为：

```text
(n - 1) / 2
```

## 12. 顺序表常用算法套路

### 12.1 双指针覆盖删除

适合删除满足条件的元素。

题目：删除顺序表中所有值为 `x` 的元素。

```c
void DeleteAllX(SeqList *L, ElemType x)
{
    int k = 0;

    for (int i = 0; i < L->length; i++)
    {
        if (L->data[i] != x)
        {
            L->data[k] = L->data[i];
            k++;
        }
    }

    L->length = k;
}
```

思路：

- `i` 扫描原表。
- `k` 表示保留元素应该写入的位置。
- 不等于 `x` 的元素保留，等于 `x` 的元素跳过。

时间复杂度：`O(n)`。

空间复杂度：`O(1)`。

### 12.2 删除满足条件的元素

例如删除所有偶数：

```c
void DeleteEven(SeqList *L)
{
    int k = 0;

    for (int i = 0; i < L->length; i++)
    {
        if (L->data[i] % 2 != 0)
        {
            L->data[k++] = L->data[i];
        }
    }

    L->length = k;
}
```

这类题的核心是“保留不该删除的元素”。

### 12.3 删除区间范围内的元素

题目：删除所有值在 `[s, t]` 范围内的元素。

```c
int DeleteRange(SeqList *L, ElemType s, ElemType t)
{
    if (s > t)
    {
        return 0;
    }

    int k = 0;

    for (int i = 0; i < L->length; i++)
    {
        if (L->data[i] < s || L->data[i] > t)
        {
            L->data[k++] = L->data[i];
        }
    }

    L->length = k;
    return 1;
}
```

适用于无序顺序表。

### 12.4 有序表删除区间范围内的元素

如果顺序表已经递增有序，可以先找到区间边界，再整体移动。

```c
int DeleteSortedRange(SeqList *L, ElemType s, ElemType t)
{
    if (s > t)
    {
        return 0;
    }

    int left = 0;
    while (left < L->length && L->data[left] < s)
    {
        left++;
    }

    int right = left;
    while (right < L->length && L->data[right] <= t)
    {
        right++;
    }

    while (right < L->length)
    {
        L->data[left++] = L->data[right++];
    }

    L->length = left;
    return 1;
}
```

要点：

- `[0, left - 1]` 是保留的前半段。
- `[left, right - 1]` 是要删除的区间。
- `[right, length - 1]` 是需要前移的后半段。

### 12.5 删除有序顺序表中的重复元素

题目：递增有序顺序表中，每个值只保留一个。

```c
void DeleteDuplicates(SeqList *L)
{
    if (L->length == 0)
    {
        return;
    }

    int k = 0;

    for (int i = 1; i < L->length; i++)
    {
        if (L->data[i] != L->data[k])
        {
            k++;
            L->data[k] = L->data[i];
        }
    }

    L->length = k + 1;
}
```

思路：

- `k` 指向已保留区域的最后一个元素。
- `i` 扫描后续元素。
- 遇到新值时，写到 `k + 1` 的位置。

### 12.6 顺序表逆置

题目：原地逆置顺序表。

```c
void Reverse(SeqList *L)
{
    int left = 0;
    int right = L->length - 1;

    while (left < right)
    {
        ElemType temp = L->data[left];
        L->data[left] = L->data[right];
        L->data[right] = temp;

        left++;
        right--;
    }
}
```

时间复杂度：`O(n)`。

空间复杂度：`O(1)`。

### 12.7 逆置指定区间

将下标 `[left, right]` 范围内的元素原地逆置。

```c
int ReverseRange(SeqList *L, int left, int right)
{
    if (left < 0 || right >= L->length || left > right)
    {
        return 0;
    }

    while (left < right)
    {
        ElemType temp = L->data[left];
        L->data[left] = L->data[right];
        L->data[right] = temp;

        left++;
        right--;
    }

    return 1;
}
```

### 12.8 合并两个递增有序顺序表

题目：已知顺序表 `A` 和 `B` 都递增有序，将二者合并成新的递增顺序表 `C`。

```c
int MergeList(SeqList A, SeqList B, SeqList *C)
{
    if (A.length + B.length > MAXSIZE)
    {
        return 0;
    }

    int i = 0;
    int j = 0;
    int k = 0;

    while (i < A.length && j < B.length)
    {
        if (A.data[i] <= B.data[j])
        {
            C->data[k++] = A.data[i++];
        }
        else
        {
            C->data[k++] = B.data[j++];
        }
    }

    while (i < A.length)
    {
        C->data[k++] = A.data[i++];
    }

    while (j < B.length)
    {
        C->data[k++] = B.data[j++];
    }

    C->length = k;
    return 1;
}
```

时间复杂度：`O(m + n)`。

空间复杂度：`O(m + n)`，因为构造了新表。

### 12.9 原地合并两个有序数组

如果 `A` 后面有足够空间，可以从后往前合并，避免覆盖。

```c
void MergeInPlace(ElemType A[], int m, ElemType B[], int n)
{
    int i = m - 1;
    int j = n - 1;
    int k = m + n - 1;

    while (i >= 0 && j >= 0)
    {
        if (A[i] > B[j])
        {
            A[k--] = A[i--];
        }
        else
        {
            A[k--] = B[j--];
        }
    }

    while (j >= 0)
    {
        A[k--] = B[j--];
    }
}
```

要点：

- 从后往前放元素。
- 如果从前往后合并，可能覆盖 `A` 中尚未处理的元素。

### 12.10 二分查找

顺序表支持随机访问，因此有序顺序表可以使用二分查找。

```c
int BinarySearch(SeqList L, ElemType target)
{
    int left = 0;
    int right = L.length - 1;

    while (left <= right)
    {
        int mid = left + (right - left) / 2;

        if (L.data[mid] == target)
        {
            return mid + 1;
        }
        else if (L.data[mid] < target)
        {
            left = mid + 1;
        }
        else
        {
            right = mid - 1;
        }
    }

    return 0;
}
```

返回值：

- 找到：返回位序。
- 未找到：返回 `0`。

时间复杂度：`O(log n)`。

### 12.11 查找插入位置

题目：在有序顺序表中找到 `target` 应该插入的位置，返回下标。

```c
int LowerBound(SeqList L, ElemType target)
{
    int left = 0;
    int right = L.length;

    while (left < right)
    {
        int mid = left + (right - left) / 2;

        if (L.data[mid] < target)
        {
            left = mid + 1;
        }
        else
        {
            right = mid;
        }
    }

    return left;
}
```

含义：

- 返回第一个大于等于 `target` 的位置。
- 如果所有元素都小于 `target`，返回 `length`。

### 12.12 将奇数移动到偶数前面

不要求保持相对顺序时，可以用双指针交换。

```c
void MoveOddBeforeEven(SeqList *L)
{
    int left = 0;
    int right = L->length - 1;

    while (left < right)
    {
        while (left < right && L->data[left] % 2 != 0)
        {
            left++;
        }

        while (left < right && L->data[right] % 2 == 0)
        {
            right--;
        }

        if (left < right)
        {
            ElemType temp = L->data[left];
            L->data[left] = L->data[right];
            L->data[right] = temp;
        }
    }
}
```

时间复杂度：`O(n)`。

空间复杂度：`O(1)`。

如果要求保持相对顺序，通常需要用覆盖法或辅助数组。

### 12.13 稳定分区

题目：保持相对顺序，将小于 `x` 的元素放前面，大于等于 `x` 的元素放后面。

使用辅助数组：

```c
int StablePartition(SeqList *L, ElemType x)
{
    ElemType temp[MAXSIZE];
    int k = 0;

    for (int i = 0; i < L->length; i++)
    {
        if (L->data[i] < x)
        {
            temp[k++] = L->data[i];
        }
    }

    for (int i = 0; i < L->length; i++)
    {
        if (L->data[i] >= x)
        {
            temp[k++] = L->data[i];
        }
    }

    for (int i = 0; i < L->length; i++)
    {
        L->data[i] = temp[i];
    }

    return 1;
}
```

时间复杂度：`O(n)`。

空间复杂度：`O(n)`。

### 12.14 顺序表循环左移

题目：将顺序表左移 `p` 位。

例子：

```text
原表：1 2 3 4 5 6
p = 2
结果：3 4 5 6 1 2
```

可以用三次逆置：

1. 逆置前 `p` 个元素。
2. 逆置后 `n - p` 个元素。
3. 逆置整个顺序表。

```c
void ReverseArray(ElemType data[], int left, int right)
{
    while (left < right)
    {
        ElemType temp = data[left];
        data[left] = data[right];
        data[right] = temp;
        left++;
        right--;
    }
}

void LeftRotate(SeqList *L, int p)
{
    if (L->length == 0)
    {
        return;
    }

    p %= L->length;

    ReverseArray(L->data, 0, p - 1);
    ReverseArray(L->data, p, L->length - 1);
    ReverseArray(L->data, 0, L->length - 1);
}
```

时间复杂度：`O(n)`。

空间复杂度：`O(1)`。

### 12.15 顺序表循环右移

右移 `p` 位等价于左移 `n - p` 位。

```c
void RightRotate(SeqList *L, int p)
{
    if (L->length == 0)
    {
        return;
    }

    p %= L->length;
    LeftRotate(L, L->length - p);
}
```

### 12.16 求两个有序顺序表的交集

```c
int Intersection(SeqList A, SeqList B, SeqList *C)
{
    int i = 0;
    int j = 0;
    int k = 0;

    while (i < A.length && j < B.length)
    {
        if (A.data[i] == B.data[j])
        {
            if (k == 0 || C->data[k - 1] != A.data[i])
            {
                if (k >= MAXSIZE)
                {
                    return 0;
                }
                C->data[k++] = A.data[i];
            }
            i++;
            j++;
        }
        else if (A.data[i] < B.data[j])
        {
            i++;
        }
        else
        {
            j++;
        }
    }

    C->length = k;
    return 1;
}
```

时间复杂度：`O(m + n)`。

### 12.17 求两个有序顺序表的并集

```c
int UnionSorted(SeqList A, SeqList B, SeqList *C)
{
    int i = 0;
    int j = 0;
    int k = 0;

    while (i < A.length && j < B.length)
    {
        ElemType value;

        if (A.data[i] < B.data[j])
        {
            value = A.data[i++];
        }
        else if (A.data[i] > B.data[j])
        {
            value = B.data[j++];
        }
        else
        {
            value = A.data[i];
            i++;
            j++;
        }

        if (k == 0 || C->data[k - 1] != value)
        {
            if (k >= MAXSIZE)
            {
                return 0;
            }
            C->data[k++] = value;
        }
    }

    while (i < A.length)
    {
        ElemType value = A.data[i++];
        if (k == 0 || C->data[k - 1] != value)
        {
            if (k >= MAXSIZE)
            {
                return 0;
            }
            C->data[k++] = value;
        }
    }

    while (j < B.length)
    {
        ElemType value = B.data[j++];
        if (k == 0 || C->data[k - 1] != value)
        {
            if (k >= MAXSIZE)
            {
                return 0;
            }
            C->data[k++] = value;
        }
    }

    C->length = k;
    return 1;
}
```

## 13. 顺序表与 C++ vector

C++ 中的 `vector` 是动态顺序表的典型实现。

常见操作：

```cpp
#include <vector>
using namespace std;

vector<int> a;

a.push_back(10);
a.push_back(20);
a.push_back(30);

int x = a[1];
a[1] = 99;
```

常见复杂度：

| 操作 | 复杂度 |
|---|---:|
| `a[i]` | `O(1)` |
| `push_back` | 摊还 `O(1)` |
| `insert` 中间插入 | `O(n)` |
| `erase` 中间删除 | `O(n)` |
| `size` | `O(1)` |

注意：

- `vector` 扩容可能导致原来的迭代器、指针、引用失效。
- 中间插入和删除会移动元素。
- 如果已知大概元素个数，可以提前 `reserve`。

```cpp
vector<int> a;
a.reserve(1000);
```

`reserve` 只改变容量，不改变元素个数。

## 14. 顺序表常见题型

### 14.1 增删查改模拟题

- 区分位序和下标。
- 插入时从后往前移动。

- 操作成功后更新 `length`。

### 14.2 原地删除题

常见描述：

- 删除有序表中的重复元素。
- 删除区间 `[s, t]` 内的元素。

int k = 0;
for (int i = 0; i < L->length; i++)

## 15. 常见错误

### 15.1 位序和下标混用

位序从 `1` 开始，下标从 `0` 开始。

```text
第 i 个元素 = data[i - 1]
下标 pos 的元素 = data[pos]
```

### 15.2 插入位置范围写错

位序插入的合法范围：

```text
1 <= i <= length + 1
```

下标插入的合法范围：

```text
0 <= pos <= length
```

注意：允许在末尾后面插入。

### 15.3 删除位置范围写错

位序删除的合法范围：

```text
1 <= i <= length
```

下标删除的合法范围：

```text
0 <= pos < length
```

注意：不能删除 `length` 下标，因为最后一个合法下标是 `length - 1`。

### 15.4 插入时移动方向错误

插入时应从后往前移动：

```c
for (int j = L->length; j >= i; j--)
{
    L->data[j] = L->data[j - 1];
}
```

如果从前往后移动，会覆盖尚未搬移的数据。

### 15.5 删除时移动方向错误

删除时应从前往后移动：

```c
for (int j = i; j < L->length; j++)
{
    L->data[j - 1] = L->data[j];
}
```

如果循环方向写反，可能死循环或得到错误结果。

### 15.6 访问元素后才判断合法性

错误写法：

```c
*e = L->data[pos];
if (pos < 0 || pos >= L->length)
{
    return 0;
}
```

正确写法：

```c
if (pos < 0 || pos >= L->length)
{
    return 0;
}
*e = L->data[pos];
```

必须先判断位置合法，再访问数组。

### 15.7 忘记更新 length

插入成功后：

```c
L->length++;
```

删除成功后：

```c
L->length--;
```

如果忘记更新长度，后续遍历、插入、删除都会出错。

### 15.8 动态顺序表忘记释放内存

动态申请的空间需要释放。

```c
free(L->data);
L->data = NULL;
```

如果结构体本身也是 `malloc` 得到的，也要释放结构体。

```c
free(L);
```

### 15.9 realloc 使用不当

错误写法：

```c
L->data = realloc(L->data, newSize);
if (L->data == NULL)
{
    return 0;
}
```

如果 `realloc` 失败，原地址会丢失，造成内存泄漏。

推荐写法：

```c
ElemType *newData = realloc(L->data, newSize);
if (newData == NULL)
{
    return 0;
}
L->data = newData;
```

## 16. 静态顺序表最小完整示例

```c
#include <stdio.h>

#define MAXSIZE 100

typedef int ElemType;

typedef struct
{
    ElemType data[MAXSIZE];
    int length;
} SeqList;

void InitList(SeqList *L)
{
    L->length = 0;
}

int ListInsert(SeqList *L, int i, ElemType e)
{
    if (i < 1 || i > L->length + 1)
    {
        return 0;
    }

    if (L->length >= MAXSIZE)
    {
        return 0;
    }

    for (int j = L->length; j >= i; j--)
    {
        L->data[j] = L->data[j - 1];
    }

    L->data[i - 1] = e;
    L->length++;
    return 1;
}

int ListDelete(SeqList *L, int i, ElemType *e)
{
    if (i < 1 || i > L->length)
    {
        return 0;
    }

    *e = L->data[i - 1];

    for (int j = i; j < L->length; j++)
    {
        L->data[j - 1] = L->data[j];
    }

    L->length--;
    return 1;
}

void PrintList(SeqList L)
{
    for (int i = 0; i < L.length; i++)
    {
        printf("%d ", L.data[i]);
    }
    printf("\n");
}

int main(void)
{
    SeqList L;
    ElemType deleted;

    InitList(&L);

    ListInsert(&L, 1, 10);
    ListInsert(&L, 2, 20);
    ListInsert(&L, 3, 30);
    ListInsert(&L, 2, 15);

    printf("插入后：");
    PrintList(L);

    if (ListDelete(&L, 3, &deleted))
    {
        printf("删除的元素：%d\n", deleted);
    }

    printf("删除后：");
    PrintList(L);

    return 0;
}
```

## 17. 动态顺序表最小完整示例

```c
#include <stdio.h>
#include <stdlib.h>

#define INIT_SIZE 10

typedef int ElemType;

typedef struct
{
    ElemType *data;
    int length;
    int capacity;
} SeqList;

int InitList(SeqList *L)
{
    L->data = (ElemType *)malloc(sizeof(ElemType) * INIT_SIZE);
    if (L->data == NULL)
    {
        return 0;
    }

    L->length = 0;
    L->capacity = INIT_SIZE;
    return 1;
}

void DestroyList(SeqList *L)
{
    free(L->data);
    L->data = NULL;
    L->length = 0;
    L->capacity = 0;
}

int ExpandList(SeqList *L)
{
    int newCapacity = L->capacity * 2;
    ElemType *newData = (ElemType *)realloc(L->data, sizeof(ElemType) * newCapacity);

    if (newData == NULL)
    {
        return 0;
    }

    L->data = newData;
    L->capacity = newCapacity;
    return 1;
}

int PushBack(SeqList *L, ElemType e)
{
    if (L->length >= L->capacity)
    {
        if (!ExpandList(L))
        {
            return 0;
        }
    }

    L->data[L->length++] = e;
    return 1;
}

void PrintList(SeqList L)
{
    for (int i = 0; i < L.length; i++)
    {
        printf("%d ", L.data[i]);
    }
    printf("\n");
}

int main(void)
{
    SeqList L;

    if (!InitList(&L))
    {
        return 1;
    }

    for (int i = 1; i <= 20; i++)
    {
        PushBack(&L, i);
    }

    PrintList(L);
    DestroyList(&L);

    return 0;
}
```

## 20. 总结

顺序表的核心是“连续存储”。

最重要的四点：

1. 顺序表支持随机访问，按位查找是 `O(1)`。
2. 插入时元素要后移，删除时元素要前移。
3. 位序从 `1` 开始，下标从 `0` 开始。
4. 静态顺序表容量固定，动态顺序表扩容需要搬移元素。

做顺序表题时，先分清题目使用的是“位序”还是“下标”，再判断操作是否合法，最后决定元素移动方向。只要这三步清楚，大多数顺序表题都能稳定写出来。
